
# WordLens

## Summary
Chrome extention in which you highlight a word, right-click, and choose Search Definition to see an inline panel with the definition.

## How-to install

### Step 1 

Download the lastest release
![Step-1](/docs/steps-imgs/step1.png)

### Step 2

- Go to `chrome://extensions`
- Press "load unpacked"
- Add the file (unzipped)

You should see something like this
![Step-2](/docs/steps-imgs/image%20(3).png)

## How to use

### Search definition

Highlight and right-click any word, go to "WordLens - Dictionary Lookup" choose "Search Definition".

![Step-3](/docs/steps-imgs/export-1782774490802.gif)
