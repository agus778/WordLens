WordLens Privacy Policy

Last updated: July 6, 2026

WordLens is a Chrome extension that lets users look up the definition of a selected English word directly from a web page.

Data Collection

WordLens does not collect, sell, share, or transfer personal user data.

Stored Data

WordLens uses Chrome storage only to save extension preferences, such as the user's custom dictionary links and selected language setting. These preferences are used only to provide and customize the extension's functionality.

Dictionary Lookup Requests

When a user explicitly selects a word and starts a dictionary lookup, WordLens sends that selected word to the Free Dictionary API at https://dictionaryapi.dev/ to retrieve the definition. WordLens does not store a history of looked-up words.

Browsing Activity

WordLens does not track browsing history, page content, clicks, keystrokes, or other browsing activity. The extension runs on web pages only to display the inline dictionary panel after the user starts a lookup.

Third Parties

WordLens includes optional links to external dictionary websites. Opening those links is controlled by the user and is subject to the privacy policies of those external websites.

Contact

For privacy questions, contact the developer through the project's GitHub repository: https://github.com/agus778/WordLens.
